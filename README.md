**Última actualización:** 7 de julio de 2026

## Visión General del Proyecto

Bunkercore es una plataforma SaaS enfocada en **seguridad y privacidad** con arquitectura **Zero-Knowledge**. El objetivo es construir un sistema completo con:
- Backend de alto rendimiento
- Aplicación Android nativa
- Frontend web/admin
- Cifrado de extremo a extremo
- Autenticación moderna (Passkeys)
- Escalabilidad y mantenibilidad desde el día 1
